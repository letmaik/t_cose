161 /* Private value. Intentionally not documented for Doxygen. This is
162  * the size allocated for the encoded protected header parameters. It
163  * needs to be big enough for encode_protected_parameters() to
164  * succeed. It currently sized for one parameter with an algorithm ID
165  * up to 32 bits long -- one byte for the wrapping map, one byte for
166  * the label, 5 bytes for the ID. If this is made accidentially too
167  * small, QCBOR will only return an error, and not overrun any
168  * buffers.
169  *
170  * 17 extra bytes are added, rounding it up to 24 total, in case some
171  * other protected header parameter is to be added and so the test
172  * using T_COSE_TEST_CRIT_PARAMETER_EXIST can work.
173  */
174 #define T_COSE_SIGN1_MAX_SIZE_PROTECTED_PARAMETERS (1+1+5+17)
